🚀 티스토리 최적화 프로그램 - 빠른 실행 가이드

===================================================

👥 초보자분들을 위한 간단한 실행 방법

1. "티스토리최적화프로그램.exe" 파일을 더블클릭하세요
2. 프로그램이 실행되면 "🔑 API 키 설정" 버튼을 클릭하세요
3. OpenAI API 키를 입력하고 저장하세요

   ➤ API 키 발급: https://platform.openai.com/api-keys
   ➤ 회원가입 후 결제 정보 등록 필요 (사용량에 따라 과금)

4. 주제를 입력하고 "🚀 일괄 생성 시작" 버튼을 클릭하세요!

===================================================

📋 프로그램 사용 시 주의사항

✅ 인터넷 연결이 필요합니다
✅ OpenAI API 키가 반드시 필요합니다
✅ API 사용량에 따라 요금이 부과됩니다
✅ 처음엔 5-10개 주제로 테스트해보세요

===================================================

🆘 문제 해결

Q: 프로그램이 실행되지 않아요
A: Windows Defender에서 차단될 수 있습니다. 
   "자세한 정보" → "실행" 클릭하여 허용하세요

Q: API 키 오류가 나와요  
A: https://platform.openai.com/api-keys 에서 
   새로운 API 키를 발급받아 다시 입력하세요

Q: 생성이 너무 느려요
A: AI 모델을 "gpt-4o-mini"로 변경해보세요
